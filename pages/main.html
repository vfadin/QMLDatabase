Справка =============== ### Главное окно В программе можно регистрировать новых пациентов. После этого содержимое полей данных становится доступным для использования в элементах управления. Для этого необходимо нажать соответствующую кнопку "Новый пациент". Будет открыто новое окно, где необходимо нажать на соответсвующие подписям поля и заполнить их, для отмены добавления используется кнопка "Отмена" для подтверждения добавления в базу "ОК" Для проведения манипуляций с полями базы данных существует контекстное меню, вызываемое нажатие правой кнопкой мыши по записи в таблице. ### Контекстное меню * Удалить
Используется для удаления записей из базы данных. По нажатию появится окно подтверждения удаления. * Редактировать
Используется для изменения данных в конкретной записи. По нажатию появится окно с формой заполнения (окно полностью аналогично окну добавления пациента). * Новый приём
Используется для добавления пациента. По нажатию клавиши появится окно "Добавление пациента". * Все приёмы
Используется для просмотра истории посещения пациента. По нажатию клавиши будет открыто окно просмотра.
